OpenLineage / Marquez
=====================

Marquez [#]_ [#]_ is an open source implementation of the OpenLineage [#]_ .


* Event specification [#]_

* Integrations (automatic collection of metadata) from Spark, Airflow, dbt [#]_

* Visualization [#]_

* Metadata API [#]_

* Column-level lineage support (so far only Spark) [#]_

* AWS Big Data Blog: Automate data lineage on Amazon MWAA with OpenLineage [#]_
